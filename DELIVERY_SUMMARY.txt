PRODUCTION DEPLOYMENT STEPS
================================================================================

1. Install Python 3.8+ on server
2. Copy all files to production folder
3. Run: pip install -r requirements.txt
4. Run: python app.py (or .\run.ps1 on Windows)
5. Access: http://[server]:8000/dashboard.html
6. Test with sample 7-file upload
7. Verify all metrics display correctly
8. Enable firewall rules for ports 8000, 5000
9. Set up database backup (daily)
10. Configure monitoring/alerts

See DEPLOYMENT_CHECKLIST.md for detailed steps.

================================================================================
WEEKLY OPERATION (5 minutes/week)
================================================================================

Monday 8:00 AM:
  1. Download 7 files from Ceipal/HRMS
  2. Go to upload.html
  3. Drag files into upload box
  4. Click "Process Files"
  5. Wait ~60 seconds for processing
  6. Go to dashboard.html
  7. Review metrics
  8. Click "Approve & Publish" if correct
  9. Share dashboard URL with team

That's it. 83 manual steps → 1 click.
