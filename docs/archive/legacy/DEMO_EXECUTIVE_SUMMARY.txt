CRITICAL DESIGN DECISIONS EMBEDDED
===================================
1. TRIM + Exact-Match Lookup
   - All AM/Director names normalized before lookup
   - Zero silent pass-throughs (hard stop on mismatch)
   - Canonical table in Config sheet (read at startup)

2. Hard Stop Policy
   - Unresolved names trigger immediate halt
   - Max retry attempts enforced (default = 3)
   - After max retries: escalation artifact (CSV) written

3. Error Recovery Flow
   - Errors tab written via Excel COM
   - Batch write (all-or-nothing, fresh each time)
   - Errors persist across retry attempts
   - Operator can fix Config sheet between retries

4. Checkpoints are Audit-Only
   - Each step writes checkpoint (timestamp, rows, status)
   - Checkpoints NEVER read for resume (always fresh re-run)
   - Purpose: post-mortem analysis, not state persistence

5. Run Folder Isolation
   - Every execution creates folder: run_YYYYMMDD_HHMMSS/
   - Contains: checkpoints/, handoffs/ (Parquet + CSV)
   - Auto-purge at end: keeps last 5 (configurable via Config)
   - No cross-run contamination
